TOPIC 4: Voice-Based Indoor Navigation Module Implementation

ASSISTANT: Implemented full voice-based indoor navigation system.

== Architecture ==

New Kotlin files created:
  1. VoiceCommandProcessor.kt  — SpeechRecognizer wrapper + rule-based NLP
  2. PathPlanner.kt            — A* path planning on occupancy grid with obstacle inflation
  3. NavigationGuide.kt        — TTS engine + turn-by-turn instruction generator
  4. NavigationManager.kt      — Full state machine orchestrating all modules

Modified files:
  1. AndroidManifest.xml       — Added RECORD_AUDIO permission
  2. ArActivity.kt             — Integrated NavigationManager, nav MethodChannel, mic button UI
  3. lib/indoor_map_viewer.dart — Nav UI: instruction banner, mic button, path rendering

== Key Technical Decisions ==

- Separate MethodChannel `com.ketan.slam/nav` for navigation (avoids overwriting
  MainActivity's handler on `com.ketan.slam/ar`)
- A* with INFLATE=1 cell safety margin; goal cell always walkable
- `needsGrid` flag prevents expensive grid copy at 30fps when nav is idle
- `observations >= 2` filter avoids navigating to single-frame YOLO hallucinations
- Heading: ARCore +X=right, -Z=forward → `atan2(dx, -dz)`, normalized to [-π, π]
- navPath sent in local grid coords, parsed in Flutter's `_parseNavPath()`
- Nav path (emerald green) drawn BEFORE BFS path (blue) so both coexist visually
- All UI callbacks wrapped in `runOnUiThread {}` for thread safety

== Navigation State Machine ==
  IDLE → LISTENING → PLANNING → NAVIGATING → ARRIVED
                                     ↓
                                  RE_ROUTING (on deviation > 2.0m)

== Voice Commands Examples ==
  "take me to the nearest door"   → {type: door, qualifier: nearest}
  "navigate to the exit"          → {type: exit, qualifier: nearest}
  "find the washroom"             → {type: washroom, qualifier: nearest}
  "go to the farthest window"     → {type: window, qualifier: farthest}

== Files Created ==

  android/app/src/main/kotlin/com/ketan/slam/PathPlanner.kt            (~177 lines)
